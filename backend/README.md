# backend

# Installation

Depending on your server address and port number being expose, you could, for example, do the following.

```
bash install.sh -s 172.24.0.1 -p 8000
```

# How to run

Go to [http://172.24.0.1:8000](http://172.24.0.1:8000)

# Requirements

- Docker
- Docker-compose
